Pedra Papel Tesoura

Este trabalho foi o primeiro que fiz que integra JavaScript com HTML. Ele foi proposto para aprender a manipular o DOM e utilizar o prompt. Os requisitos eram: utilizar o prompt para o jogador selecionar suas 3 opções as digitando e realizar uma "melhor de três" contra o computador. Para o jogador selecionar suas 3 opções, ele deve escrever "pedra", "papel" ou "tesoura" tanto em maiúsculas quanto minúsculas que o sistema vai aceitar. Se o jogador tentar enviar qualquer outra palavra, o sistema acusa que aquela opção é inválida e só passa para a próxima opção quando uma opção válida é inserida.

Rock Paper Scissors

Após ter feito o exercício anterior, decidi fazer um jogo sem ter a "melhor de três" e que o jogador escolhe sua opção clicando na imagem que representa "pedra", "papel" ou "tesoura". Cada vez que o jogador clica, selecionando assim sua opção, é mostrada qual foi a opção do computador e do jogador, se o jogador ganhou ou perdeu e mantém um placar com as pontuações de ambos.
